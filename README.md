## 💸 API RESTful de Transferências Financeiras (Estilo PicPay)
**Tecnologias:** Java • Spring Boot • REST • JPA

Sistema backend simulando uma aplicação de pagamentos digitais, aplicando regras reais do mercado financeiro.

**Principais funcionalidades:**
- Transferências entre usuários comuns e lojistas
- Controle de permissões por tipo de usuário (enum)
- Validações de negócio:
  - Unicidade de CPF e e-mail
  - Verificação de saldo
  - Bloqueio de transferências inválidas
- Uso de transações (`@Transactional`) para garantir consistência financeira
- Integração com serviço externo de autorização de transferências
- Envio de notificações assíncronas (mock de serviço externo)
- Tratamento global de exceções com respostas HTTP padronizadas
- Estrutura organizada em Controller, Service, Repository, DTOs e Entities
- Testes completos via Postman
